title: Images
description: Manage the shared image library used across content and proposals.
---

Ratiba stores every uploaded image in a workspace-wide image library. The same image can be attached to multiple accommodations, activities, or proposal hero slots.

## Uploading

You can upload from any of these places:

- The image picker on an accommodation or activity.
- The dedicated **Content Library → Images** view, for bulk upload.
- Drag-and-drop directly into the proposal hero slot.

Supported formats: JPEG, PNG, WebP, GIF. Max file size 20 MB per image. We recommend at least 1600 px on the long edge for high-quality proposal headers.

## Storage and CDN

Images are stored in your workspace's bucket and served from a CDN. URLs are stable once issued, so you can also reference them externally if needed.

## Bulk upload

In the Images view, drag a folder of photos onto the page to upload them in one batch. Each image gets a default name from the filename, which you can edit afterward.

## Captions and alt text

Each image has optional caption and alt-text fields. Captions appear under the image in proposals. Alt text is used for accessibility and SEO when proposals are indexed.

## Removing an image

Removing an image from the library deletes the file. Any accommodation or activity that referenced it will lose that image. You'll be warned before deletion.

If an accommodation needs to drop an image but you want to keep the file, just remove the link, don't delete the file.
